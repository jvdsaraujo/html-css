Citações simples e citações completas

Para utilizar citações basta apenas usar a tag q, como diria o Jake Alerta de ciclone rapaziada, todos os folgados para casa imediatamente!.

Para utilizar citações completas, podemos utilizar a tag blockquote, para que existe uma quebra de patagrafo numa citação maior, tal qual o generico ipsum lorem dizia:

"Lorem ipsum dolor sit amet consectetur adipisicing elit. Quidem quas ut aliquam dolor sit. Nostrum ab, unde, repudiandae sunt error perspiciatis, atque beatae aspernatur fugiat sed velit aperiam fuga asperiores."
Na blockquote ainda existe o mecanismo de você colocar o elemento cite dentro dela, e linkar o endereço de onde você pegou a citação. Pros mecanismos de busca do google mostrarem o seu site se alguem estiver procurando o assunto que foi citado por você ali, ou procurarem sobre o livro também.
