Texto marcado

Para marcar um texto, utilizamos a tag mark para que fique assim. Para que o texto fique com a cor de um marcador diferente, basta usar um padrão de CSS com style e background color, dai o texto fica dessa forma.MUITO FORTE.

Texto grande e texto pequeno

Para texto grande temos a tag Big, mas ela esta obsoleta, então teríamos que ver no CSS qual estilo usar em seu lugar. Para texto pequeno temos a tag small que ainda está em uso pelo HTML5.

Texto deletado e texto inserido

Para texto deletado temos a tag DEL que o próprio nome já é um pouco auto explicativo. Para textos inseridos, ou seja, textos importantes que você quer que o seu cliente leia, você tem a tag INS, ou seja, texto INSERIDO.

Texto sobescrito e texto Subescrito
