Testando carga e imagens

Abaixo você verá imagens que provem de uma pasta dentro da pasta de exercícios

Para utilizar uma imagem da propria pasta, basta apenas usar a tag de img com um caminho diferente

Para utilizar uma imagem que esteja em outro local do seu computador ainda, numa subpasta etc.

Para utilizar uma imagem de outro servidor, basta colocar o endereço dela
