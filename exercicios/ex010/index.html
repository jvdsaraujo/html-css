Links

Links externos

Para criar links externos, basta dar um ctrl+c ctrl+v na tag <a> e ta pronto. Se não acredita em mim, clique aqui e prove o meu ponto!

Links internos

Para criação de links internos, basta utilizar a mesma tag, mas ao invés de colocar o link externo, basta colocar o caminho de uma pagina html existente dentro da pasta do seu codigo.

Para ir para a terceira página direto do meu site, basta clicar aqui.

Links para download

Para criar um link de download a partir de um pdf basta mudar a <a>. Dessa forma:

  • Baixar o livro em PDF
  • Baixar o arquivo compactado
